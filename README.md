# Cześć Maria

Przygotowaliśmy dla Ciebie środowisko z wykorzystaniem https://create-react-app.dev/
Z CV widzimy, że lubisz psy zatem do realizacji zadania wykorzystaj publiczne api dostępne tutaj: https://dog.ceo/dog-api/documentation/

W ramach zadania chcielibyśmy, abyś wypełniła poniższe kryteria akceptacji:

- aplikacja prezentuje listę rozwijaną typu select, służącą do filtrowania Twoich ulubionych ras psów,
- domyślnie powinna być wybrana pierwsza pozycja - Najlepsze zdjęcia - która na początku jest pusta,
- poniżej listy select prezentowanych jest 5 losowych zdjęć wskazanej rasy,
- każde zdjęcie powinno posiadać opcję zapisania do 'Najlepsze zdjęcia',
- wybranie z listy 'Najlepsze zdjęcia' prezentuje zapisane zdjęcia bez odpytania api,
- napisany przez Ciebie kod posiada testy jednostkowe,
- tworząc komponenty załóż, że będziemy chcieli reużyć ich w przyszłości, w innych miejscach aplikacji

Kwestie wizualne nie są dla nas na tym etapie najważniejsze, nie musisz tworzyć żadnych styli.
